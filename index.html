Pane

Remote dev environments on macOS & iOS

Connect to your servers over SSH. Organize projects into workspaces. Split terminals, run services, browse files, view diffs — all from a single native app. Start on your Mac, pick up on your iPhone.

Download for macOS

Requires macOS 15 or later. Updates automatically via Sparkle.

Ghostty Terminal

GPU-accelerated terminal rendering with Metal. Full xterm-256color, split panes, and tmux-backed sessions that persist across restarts.

Project Workspaces

Organize servers into projects and workspaces. Branch workspaces create git worktrees. Layout syncs across every connected device.

Services

Declare dev servers and workers in pane.json. Start, stop, and health-check them from the sidebar. Open service URLs in a built-in browser routed through the server.

AI Agents

Launch Claude Code or any CLI agent in a dedicated tab. Get attention pulses when agents need input. Auto-generated tab titles from conversation context.

Remote Browser

Browse localhost and internal URLs on remote servers. All traffic routes through a SOCKS proxy on the host — no port forwarding needed.

iOS

Full terminal on your iPhone with the same Ghostty engine and tmux sessions. Speech input, file browser, and seamless handoff between devices.
